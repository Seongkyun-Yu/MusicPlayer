# MusicPlayer

![musicplayer](https://user-images.githubusercontent.com/15887982/78346812-8d2bce00-75da-11ea-875d-9c2e322289cc.png)

<br>

## 조원
- 허용제
- 유성균
- 윤관영

- 곡 변경시 제목, 작곡가, 커버 변경
